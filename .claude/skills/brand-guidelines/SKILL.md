# Family Planner Brand

## Colors

- **Primary**: `#13ec92` (CTAs, accents)
- **Background**: `#f6f8f7` light / `#10221a` dark
- **Surface**: `#ffffff` light / `#1c2e26` dark
- **Text**: `#111815` light / `#ffffff` dark
- **Text Secondary**: `#618979` / `#8baea0`
- **Border**: `#dbe6e1` / `#2a3831`

## Typography

- **Display**: Lexend (headings, UI)
- **Body**: Noto Sans (content)

## Event Colors

Left border (4px) + tinted background:

- Blue: sports/activities
- Purple: personal/gym
- Orange: lessons
- Green: family/meals
- Red: special dates
- Yellow: celebrations

## Icons

Material Symbols Outlined. Use `icon-filled` class for filled variant.

## Animation

`transition: all 200ms ease` + `active:scale-95` for buttons.
